Behind the Scenes

The Admin Dashboard

To manage the platform effectively, I designed a comprehensive web-based admin panel. It provides a bird's-eye view of platform health, user activity, and financial metrics.

  • Listener Approval

    Review documents and activate listener profiles.

  • Wallet Management

    Monitor recharges, refunds, and listener payouts.

  • Live Monitoring

    Track active calls and server status in real-time.

Admin Dashboard UI Concept

Crafting the "Safe Space" UI

Total Anonymity

Designed a login flow that requires zero personal details. Just an OTP. The UI reinforces privacy with soothing colors and "incognito" visual cues.

Listener Connection

Listener profiles focus on empathy tags (e.g., "Good Listener", "Relationship Advice") rather than just credentials, making them feel approachable.

Transparent Wallet

A clear, stress-free "Pay per minute" wallet UI. Users always know their balance, removing anxiety about hidden costs during vulnerable moments.

Visual Identity

Soft gradients and rounded typography create a calming atmosphere.
